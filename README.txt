#Инфо
Плагин: InterKassa Gateway
Описание: Платежный шлюз "Интеркасса" для сайтов на WordPress. (версия Интеркассы 2.0)
Предназначение: Модуль добавляет способ оплаты "Интеркасса" для плагина WooCommerce 2.5.5+, Версия движка 4.5.X
Версия: 1.4
Автор: Gateon
Сайт автора: http://www.gateon.net

#Установка
1. Скачать плагин к себе на компьютер, распаковать
2. Закачать папку с плагином на сервер с сайтом в папку /wp-content/plugins
3. зайти в админку, найти в списке плагин "InterKassa Gateway" и нажать "Активировать"
4. Перейти на вкладку настроек WooCommerce/вкладка Оплата(Checkout)/ Интеркасса 2.0
5. Включить плагин, ввести данные своего кошелька с аккаунта Интеркассы, уникальный ключ, номер кошелька, заголовок и описание метода оплаты для отображения в списке оплат вашего магазина

Жмем "Сохранить изменения"

Плагин активен и появится в списке оплат вашего магазина.

#Тестовый режим для отладки модуля
1. Добавить в настройках кошелька на сайте интеркасса вид оплаты "Тестовый режим" (валюта XTS)
1. Вписать тестовый ключ из настроек кассы
2. Активировать тестовый режим модуля
3. жмем "Сохранить"

В личном кабинете в настройках "Интеркассы", раздел "Интерфейс" укажите во всех полях урл:
http://your_site_name/?wc-api=WC_Gateway_Interkassa
